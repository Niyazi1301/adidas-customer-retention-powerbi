
# Adidas Customer Retention Analysis (Power BI)

## 📊 Project Overview
This project focuses on analyzing customer retention and churn behavior for Adidas using Power BI. The objective is to identify key factors affecting customer churn and provide actionable insights to improve retention and engagement.

## 🛠️ Tools & Technologies
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)
- Excel (Data Source)

## 📂 Dataset Description
The project uses multiple datasets including:
- Customer Data (demographics, membership details)
- Transaction Data (purchase history)
- Store Data (store type, region)
- Loyalty Program Data (points earned and redeemed)

## 🔍 Key Analysis Performed
- Churn Rate Analysis by Region, Channel, and Loyalty Tier
- Customer Segmentation (Repeat, Churned, High Value)
- Purchase Frequency Analysis
- Promotion & Loyalty Impact Analysis
- Customer Lifetime Value (CLV) Analysis
- Store & Channel Performance Insights

## 📈 Key Insights
- Overall churn rate is high (~49%), indicating a major retention challenge.
- Elite and high-income customers show higher churn rates.
- Online channel has slightly higher churn compared to store purchases.
- Loyalty program engagement is low, especially in terms of point redemption.
- Store type has minimal impact on churn, but flagship stores show slightly higher attrition.

## 📊 Dashboard Features
- Interactive filters (Region, Loyalty Tier, Income Level, Store Type)
- KPI cards (Churn Rate, Total Customers, Repeat Customers, CLV)
- Visual breakdown of churn across different dimensions
- Customer segmentation dashboard
